### Requerimentos del sistema:
- node 16.x
- npm 6 o superior
- git

### Correr proyecto
- 1. Clona y descarga el repo
- 2. Ve a la raiz del proyecto y ubicate ahí dentro de una terminal
- 3. Instala dependencias con "npm i"
- 4. Ejectua el comando "npm run dev"
